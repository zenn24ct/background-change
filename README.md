背景色チェンジ！！

ワンクリックでページの背景色がランダムに変わるシンプルなWebページです。
疲れている日でも気軽に遊べる、ミニプロジェクト。

📁 構成

index.html

HTML1枚で完結

ボタンを押すだけで背景色が変わる

🎮 使い方

ブラウザで index.html を開く

「色を変える！」をクリック

背景色がランダムに変わります

🧩 主な機能

ランダムカラーを生成して背景に適用

ワンクリックで反応するシンプルUI

軽くてどのブラウザでも動作

🔧 改善・追加できそうなこと

パステルカラー限定モード

3色の中から選ぶモード（落ち着く色だけなど）

自動で色が変わるアニメーションモード

お気に入り色の保存機能

ボタンをもっと可愛くデザイン

背景にグラデーション追加（オン・オフ切り替え）
